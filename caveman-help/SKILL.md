## OUTPUT

Print this reference card:

```
━━━ CAVEMAN MODE REFERENCE ━━━

ACTIVATE
  /caveman          → full mode (default)
  /caveman lite     → remove filler, keep grammar
  /caveman full     → drop articles, use fragments
  /caveman ultra    → max compression + abbreviations
  /caveman wenyan   → classical Chinese mode
  "talk like caveman" / "less tokens please"

DEACTIVATE
  "stop caveman" / "normal mode"

INTENSITY
  lite    ~30% savings  — no filler, normal grammar
  full    ~65% savings  — fragments, no articles
  ultra   ~75% savings  — abbreviations, arrows, tables

SKILLS
  /caveman-commit   → terse Conventional Commits (≤50 chars)
  /caveman-review   → one-line PR comments
  /caveman-help     → this card
  /caveman:compress → compress memory files (~46% savings)

ALWAYS NORMAL
  Code blocks, commits, PRs, security warnings

EXCEPTIONS
  Irreversible actions (git push, rm) → ask first
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
